# Projekt z zajęć projektowych z przedmiotu "Projektowanie i Montaż elektroniki"  
English below.  

Jest to projekt płytki PCB wykonany na potrzeby zajęć projektowych z przedmiotu "Projektowanie i Montaż Elektroniki"  
  
Zajęcia te polegały na zaprojektowaniu prostej pływki PCB, a następnie samodzielnym jej wytrawieniu.  
  
Zgodnie z zaleceniami prowadzącego projekt ten bazuje na podstawie jednego z kit-ów dostępnym w sklepie AVT.  
  
Jest to układ zegara z wyświetlaczem siedmiosegmentowym. Płytka ta wykorzystuje układ scalony PCF8583 który odpowiada za funkcje zegara. Oraz mikrokontroler Atmega8, który odpowiada za obsługę wyświetlacza siedmiosegmentowego.  
  
Również zgodnie z jednym z wymogów prowadzącego, oba schematy wykonane zostały w Autodesk Eagle.  

English:  
This project was created as part of a design course focused on electronics design and assembly.  
  
The course involved designing a simple PCB and then manually etching it.  
  
According to the instructor’s guidelines, the project is based on one of the kits available in the AVT store.  
  
The system is a clock with a seven-segment display. The PCB uses the PCF8583 integrated circuit, which is responsible for timekeeping functions, and an Atmega8 microcontroller, which handles control of the seven-segment display.  
  
In accordance with the course requirements, both schematics were designed using Autodesk Eagle.
